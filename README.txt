ระบบ PsychoMVC Version 1.0.0 โดย Twiceworld PHP (nutterrocker)
วิธีใช้
1.นำไฟล์นี้ไปวางไว้ในโฟเดอร์ของคุณ
2.ไปที่ app/config/base.php
3.รันด้วย notepad++ หรือ texteditor ide ตัวไหนก็ได้
4.แก้ข้อความตาม comment เช่น base_url base_path และ folder_name
5.รันเว็บไซต์เลยโดยการเข้า
localhost/โฟเดอร์/

เรียกใช้ Controller ในกรณีไม่ได้ใช้ mod_rewrite
localhost/โฟเดอร์/index.php/controller
